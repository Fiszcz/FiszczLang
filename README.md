# FiszczLang
A new amateurish programming language 

### Example fragment of code

```
int some_number 5;
real float_number 5.0;
print some_number float_number;

read some_number;
print some_number;
print some_number + 5 * 6 - 7 / 7;

some_number 10;
print some_number;

int other_number 20;
some_number other_number;
print some_number;
```


### Concept of language

##### Types

- ``int`` - integer numbers
- ``real`` - float numbers

##### Input/Output operations

- ``read`` - read data from **stdin**
- ``print`` - print data to **stdout**

##### Arithmetic Operators

- ``+`` - plus
- ``-`` - minus
- ``*`` - multiplication
- ``/`` - division
